# HEIC to JPG Converter

Pythonで作成された、高機能なHEIC画像一括変換ツールです。
直感的なGUI操作で、複数のHEICファイルを高品質なJPEG形式へ簡単に変換できます。

## 主な機能

* **一括選択・変換**: 複数のファイルを一度に選択し、バッチ処理が可能です。
* **連続動作**: 変換完了後もアプリを閉じず、続けて別のファイルの変換作業を行えます。
* **詳細な書き出し設定**:
    * **画質調整**: 1〜100の範囲でJPEG圧縮率を指定可能。
    * **メタデータ保持**: 写真のEXIFデータ（撮影日時、位置情報など）の維持・破棄を選択可能。
* **ユーザーフレンドリーな設計**:
    * 変換中の進捗をプログレスバーでリアルタイム表示。
    * マルチスレッド処理により、変換中もGUIがフリーズしません。

---

## 必要条件

実行には以下のライブラリが必要です。

```bash
pip install Pillow pillow-heif

```

---

## 使い方

1. **ファイルの読み込み**
「HEICファイルを選択」ボタンをクリックし、変換したいファイルを複数選択します。
2. **設定の調整**
必要に応じて、保存時の「画質」や「EXIFデータの保持」を設定します。
3. **変換の実行**
「変換を開始」ボタンをクリックし、保存先のフォルダを選択すると処理が始まります。
4. **継続利用**
処理が終わると通知が表示されます。そのまま新しいファイルを選択して変換を続けることができます。

---

## 注意事項

* 保存先のフォルダに同名のファイルがある場合、上書きされる可能性がありますのでご注意ください。
* HEICファイルのサイズや枚数によっては、処理に時間がかかる場合があります。
